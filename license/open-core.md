# Open-core model

ContractShield follows an **open-core** licensing model.

## What is open-source (Core)

The following components are released as open-source:

- Policy Decision Point (PDP) core
- Policy language (CEL)
- RequestContext and Decision schemas
- Canonicalization logic
- Node.js and Java PEP adapters
- Policy packs (community)
- Golden tests runner
- Documentation and quick starts

These components are licensed under **Apache License 2.0**.

The open-source core is:
- production-usable
- auditable
- free to use commercially
- free to modify and redistribute (under Apache 2.0 terms)

No artificial limitations are imposed on the core.

---

## What is commercial

Commercial components are **not required** to run ContractShield core.

They include advanced runtime and enterprise features such as:
- sink-aware RASP enforcement
- advanced egress control
- certified policy packs
- dashboards and UI tooling
- audit exports and compliance tooling
- SLA and enterprise support

Commercial components are distributed under a proprietary license.
