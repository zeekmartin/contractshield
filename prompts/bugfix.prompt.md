# bugfix.prompt.md

You are fixing a bug in the policy layer.

## Constraints
- Minimal change that fixes the root cause.
- Add a regression test that fails before and passes after.
- Do not weaken security defaults.

## Required output
1. Reproduction steps (input context/request).
2. Root cause analysis.
3. Fix plan.
4. Patch summary.
5. Tests added.
6. Any rollout notes.
